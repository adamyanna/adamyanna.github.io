# Modern Operating Systems

> 2020-04-20

Core principles of Unix/Linux operating systems and source-level analysis.

A **namespace** in computer science (sometimes called a _name scope_) is an abstract container or environment created to hold a logical grouping of unique identifiers or symbols (i.e., names). An identifier defined in a namespace is associated only with that namespace.

In the Linux kernel, namespaces are the fundamental building block of container isolation — PID namespaces, network namespaces, mount namespaces, UTS namespaces, IPC namespaces, user namespaces, and cgroup namespaces each isolate a specific resource class.
