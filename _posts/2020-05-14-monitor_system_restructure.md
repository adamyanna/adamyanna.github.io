# 监控系统1.5版本项目重组

* 采集数据处理接口添加逻辑：
 
```python
"""
DES:
 监控项目数据处理入口
 
 @API
 API = {
 "/monitor_upload/snmp/": MonitorUploadAPI,
 }
 Key: 父级路由
 Value: 父级路由对应的子路由
 
 @MonitorUploadAPI
 MonitorUploadAPI = {
 "xxx": {"metric": "x.xx", "handler": metric.xx.xxx}
 }
 Key: 子路由
 Value: 监控项及处理函数
 metric: 监控项名称
 handler: 数据处理函数, 处理函数如参必须包含： metric 和 data
"""
```
