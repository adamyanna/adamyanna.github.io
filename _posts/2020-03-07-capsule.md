# Linux

###### 1. proc目录下cpuinfo检查

1. 查看 CPU 物理个数
　　`grep 'physical id' /proc/cpuinfo | sort -u | wc -l`

2. 查看 CPU 核心数量
　　`grep 'core id' /proc/cpuinfo | sort -u | wc -l`

3. 查看 CPU 线程数
　　`grep 'processor' /proc/cpuinfo | sort -u | wc -l`

4. 查看 CPU  型号
　　`dmidecode -s processor-version`

5. 查看 CPU 的详细信息：
　　`cat /proc/cpuinfo`

###### 4. 在ls中列出文件的绝对路径

`ls | sed "s:^:`pwd`/:"`

```sh

sed 's/$/string/g' file

find  $PWD -maxdepth 1  | xargs ls -ld
```
